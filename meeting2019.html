From InterPro:

The software:

  • Current version: InterProScan 5.35-74.0
  • Requirements:
    • 64-bit Linux
    • Perl 5 (default on most Linux distributions)
    • Python 3 (InterProScan 5.30-69.0 onwards)
    • Oracle's Java JDK/JRE version 8 (InterProScan 5.17-56.0 onwards)
  • Supports master/worker parallelisation mode
  • Web Production runs InterProScan as a web service

Data Warehouse

Release Procedure
Release Procedure
  • The procedure takes around 3-4 days
  • Requires some legacy procedure from InterPro6 (1.5 days)
  • Runs on the EBI cluster - Easy to migrate
  • It is controlled with a python App.
    • It can runs all the steps at once.
    • It can run selected parts
  • Some numbers for InterPro 74.0:
    • Proteins in MySQL: 155M
    • Documents in ElasticSearch: 1.4B
    • Ebi-search: 300K entries and 3.2B X-references
